About the Role

We're looking for a talented Senior Full Stack Developer to join our innovative team. You'll be working on cutting-edge web applications that serve millions of users worldwide. This role offers the opportunity to work with modern technologies, collaborate with a talented team, and make a real impact on our product.

Requirements / Job Description

    Grafana Administration:
  • Design, implement, and maintain Grafana dashboards to monitor infrastructure and application performance.
  • Collaborate with development and operations teams to define key metrics and build actionable dashboards.
    • EKS Cluster Management:
  • Manage Kubernetes clusters on AWS using EKS.
  • Implement and manage Kubernetes networking, storage, and security components.
    • Monitoring and Alerting:
  • Develop and maintain monitoring and alerting systems using Grafana, Prometheus, and other tools.
  • Implement alerting mechanisms to ensure timely response to performance issues and outages.
  • Analyze logs and metrics to identify and resolve performance bottlenecks and reliability issues.
    • Automation and Scripting:
  • Maintain automation scripts using python or other Infrastructure as Code (IaC) tools.
  • Automate routine tasks to improve operational efficiency and reduce manual intervention.
    • Collaboration and Support:
  • Work closely with DevOps, SRE, and development teams to provide guidance on best practices for monitoring.
  • Participate in on-call rotation to support EKS clusters and monitoring infrastructure.
